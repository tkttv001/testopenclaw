# Advisor Intake Lean

Ask only 8 required fields, then stop:
1) objective
2) target audience
3) style
4) available assets
5) must-have features
6) deadline
7) budget
8) domain/hosting status

Generate:
- Scope (in)
- Out-of-scope
- Acceptance criteria
- Open questions (max 3)

Keep messages short. Avoid repeated confirmations.
