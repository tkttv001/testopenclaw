# Defaults for 3-Question Personal Website Delivery

## Design system defaults
- Typography: strong heading + readable body
- Spacing scale: 4/8/12/16/24/32
- Radius: 12/16 for cards, 999 for pills/buttons
- Motion: subtle (200-350ms), avoid heavy parallax

## Conversion defaults
- Primary CTA always visible above fold
- Sticky CTA on mobile for contact/book call
- Proof block before CTA (experience, selected clients/projects)

## Project card defaults
- Problem -> Solution -> Outcome
- Include one measurable metric if available; else qualitative impact

## Missing information policy
If data is missing, use placeholders marked clearly as replaceable and continue delivery.
Do not block production on non-critical content gaps.
