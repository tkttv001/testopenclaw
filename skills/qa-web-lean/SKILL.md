# QA Web Lean

Checklist:
- responsive mobile/tablet/desktop
- broken links/form basic behavior
- console major errors
- visual clarity/readability

Output:
- Verdict: PASS/REJECT
- Findings (bullet)
- Required fixes (if any)
- Retest steps

Do not over-test non-MVP features.
