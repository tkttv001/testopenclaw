# QUALITY_IDEAS_3Q.md

Mục tiêu: đầu ra chất lượng cao dù chỉ hỏi 3 câu.

## Product ideas để tăng chất lượng mặc định

1. **Template theo persona**
- Developer portfolio
- Coach/Consultant
- Creative/Designer
=> Mỗi persona có layout + copy skeleton riêng.

2. **Hero quality engine**
- Tự sinh 3 headline + 2 subheadline
- Auto chọn bản mạnh nhất theo CTA clarity.

3. **Proof-first structure**
- Luôn đặt credibility block sớm: years, stack, selected projects.

4. **Auto SEO pack**
- title/description/OG/robots/sitemap/schema tự sinh theo input 3Q.

5. **Mobile polish preset**
- Font scale preset
- Touch target preset
- Sticky CTA preset

6. **Project storytelling format**
- Problem -> Solution -> Outcome (ngắn gọn, dễ quét)

7. **Content confidence policy**
- Nếu thiếu data: dùng placeholder có nhãn `[replace]` thay vì chờ khách.

8. **A/B CTA preset**
- CTA1: Contact me
- CTA2: Book a quick call
=> chọn theo audience (recruiter vs client).

9. **Fast QA gate (5 checks)**
- responsive
- no major console errors
- CTA works
- SEO tags present
- production build passes

10. **Handover-ready package**
- source + deploy note + replace-content guide
- giúp khách dùng ngay mà không hỏi thêm.

## Rule giảm hỏi lại khách hàng
- Chỉ hỏi thêm nếu thiếu thông tin làm sai mục tiêu cốt lõi (goal/audience/style).
- Mọi thứ còn lại dùng default thông minh + bàn giao editable.
