# TASK-20260220-002 — Chef Portfolio (Premium Fine Dining)

## Intake summary (locked)
1. Client profile:
- Professional chef with 6 years experience
- 3 years in USA + 3 years in Vietnam

2. Selected style:
- Option #2: **Premium Fine Dining**

## Project objective
Create a premium, visually rich, production-ready personal chef website that communicates culinary authority, cross-cultural story depth, and high-end private dining positioning.

## Primary audience
- High-net-worth private clients seeking private chef experiences
- Boutique hospitality partners and event curators
- Media/collaboration inquiries

## Conversion goals
- Primary CTA: Book a private dining consultation
- Secondary CTA: Request seasonal tasting menu / partnership inquiry

## Information architecture (5 pages)
1. Home
2. About
3. Signature (Projects/Menu equivalent)
4. Stories (Blog equivalent)
5. Contact

## Visual direction — Premium Fine Dining
- Palette: charcoal black, ivory, warm gold, muted wine accents
- Typography: editorial serif display + elegant sans body rhythm
- Imagery: cinematic plating, kitchen craft details, moody low-light scenes
- Layout language: generous whitespace, refined grid, tactile card surfaces
- Motion: subtle reveal/hover transitions only

## Content pillars
- Chef identity and philosophy (USA x Vietnam influence)
- Signature tasting experiences and sample menus
- Story-led journal content (ingredients, travel, technique)
- Premium booking pathway

## Technical requirements
- Next.js App Router + TypeScript + Tailwind
- True multi-page routes (not one-page anchors)
- Mobile-first behavior with accessible nav
- SEO baseline:
  - Metadata (global + per-page)
  - Open Graph
  - robots.ts
  - sitemap.ts
- Static export compatibility for GitHub Pages deployment

## Out of scope
- Payments / booking backend integration
- CMS integration
- Real-time availability calendar

## Acceptance criteria
- 5 routes implemented and exported successfully
- Premium fine-dining visual quality across desktop/mobile
- QA pass for navigation, responsiveness, SEO, and build
- Public preview deployed under next docs task folder
