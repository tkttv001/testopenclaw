# Desktop prompt set (5 pages)

Generate 5 separate desktop screens (2560 width) for a premium personal portfolio in **Ronin Premium Dark** style:

1. **Home**: Hero, value proposition, CTAs, featured projects teaser, latest blog teaser.
2. **About**: Story, principles, timeline, skills/approach blocks.
3. **Projects**: Case study grid, results-first cards, tag filters visual.
4. **Blog**: Content-heavy layout with post cards, dates, reading depth hierarchy.
5. **Contact**: Lead-in copy + form flow (name/email/details), urgency note.

Design constraints:
- Charcoal/dark background, subtle brush texture, gold accents.
- 8px spacing rhythm.
- Clear top nav for all pages.
- Strong typographic contrast; headings bold, body readable.
- Reusable card and button styles across all screens.
