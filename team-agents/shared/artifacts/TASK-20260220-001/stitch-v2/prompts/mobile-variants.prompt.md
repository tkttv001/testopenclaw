# Mobile-focused prompt set (390 baseline)

Generate mobile variants for:
1. **Global nav + hero** (Home mobile): collapsed menu button, stacked hero copy, dual CTAs.
2. **Content-heavy page** (Blog mobile): readable typography, card stacking, date/title/excerpt hierarchy.
3. **Contact flow mobile**: single-column form with reachable inputs, clear submit CTA, validation affordance.

Required behavior in design:
- Touch targets >=44px height.
- Off-canvas or dropdown nav interaction state represented.
- Safe spacing at 390 width (no horizontal overflow).
- Form labels always visible.
- Preserve Ronin dark aesthetics without reducing contrast.
