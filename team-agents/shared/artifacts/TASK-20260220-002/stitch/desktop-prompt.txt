- Sticky top nav with links: Home / About / Signature / Stories / Contact
- Strong hero with signature dish image, chef positioning statement, CTA buttons
- Section rhythm with generous whitespace and modular cards

Generate 5 pages:
1) Home

   - CTA block
2) About

3) Signature (Menu/Projects equivalent)

4) Stories (Blog equivalent)
   - Editorial card grid for chef notes and ingredient stories
5) Contact

UI constraints:
- WCAG-friendly contrast
- Reusable components (button variants, cards, section headers)
- Subtle motion only (fade/slide micro-interactions)
- Production-ready for React/Next implementation
- Avoid lorem ipsum; use realistic culinary copy in English
